# Titanic Survival Prediction using Logistic Regression

## Overview

This project explores the Titanic passenger dataset to identify the factors that influenced survival during the disaster. Through exploratory data analysis, feature engineering, and logistic regression, the project develops a predictive model while examining how demographic and socioeconomic variables affected passenger survival.

The project demonstrates the complete data analysis workflow, from preprocessing and visualization to model development and interpretation.

---

## Background

The Titanic dataset is one of the most widely used datasets in introductory machine learning and data science. It contains demographic and passenger information such as age, sex, ticket class, fare, and family relationships, making it suitable for binary classification problems.

Logistic regression is used to estimate the probability that a passenger survived based on these explanatory variables.

---

## Objectives

This project aims to:

* Explore patterns within the Titanic dataset.
* Perform data cleaning and preprocessing.
* Analyze relationships between passenger characteristics and survival.
* Build a logistic regression model to predict survival.
* Interpret the influence of different variables on prediction outcomes.

The dataset contains passenger information including:
